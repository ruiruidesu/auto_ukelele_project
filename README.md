# 🎸 Auto Ukulele Tab Generator

## 🌿 Project Overview

有没有过这种时刻：  
听到一首很喜欢的歌，很想马上拿起尤克里里弹出来。  
可一搜——没有谱。

这个项目就是为了解决这件事。

只需要输入一段音频，它就能自动分析旋律、节奏和和弦，  
生成适合尤克里里的指弹谱或弹唱谱。

不需要乐理基础。  
只要你听得到，你就可以弹出来。

---

## 💡 Why This Project

我自己有一定的尤克里里基础，但一直卡在“扒谱”这一步。  
很多喜欢的歌没有现成谱子，而传统扒谱对普通人门槛太高。

所以我在想：  
👉 能不能让机器帮我们“听懂音乐”？

---

## ⚙️ Features

- 输入音频（mp3 / wav）
- 自动提取旋律
- 节奏 / BPM 识别
- 和弦检测（用于弹唱）
- 转换为尤克里里谱
- 输出结构化谱子

---

## 🚧 Status

这是一个仍在迭代中的项目。  
目前还在优化准确率和复杂音频处理能力。
目前已经能够实现将尤克里里单音演奏视频输入即可输出准确音节尤克里里弹奏谱子
